# Contributing

Thank you for your interest in contributing to this project!

Currently, this repository serves as a personal portfolio project and **does not accept external Pull Requests**. I (the repository owner) am the only one actively making changes to the main codebase.

However, since this repository is public and uses an MIT license:
- You are more than welcome to **fork** the repository.
- You can use the code for your own projects, experiments, or learning.
- You can freely study the architecture and implementation details.

If you find a bug or have a suggestion, feel free to open an Issue, but please note that direct PRs to the main repository will not be merged at this time.
